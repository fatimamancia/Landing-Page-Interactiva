# Landing-Page-Interactiva
Implementación de interactividad avanzada: manipulación del DOM, validación de formularios y publicación con control de versiones y apoyo de IA. 

#🍏 Gastronomía Salvadoreña - SaborSV 🍏

Landing page desarrollada como proyecto académico para promover
la gastronomía tradicional de El Salvador. 😋

##Tecnologías utilizadas 
- HTML5
- CSS3 (Flexbox y Grid)
- JavaScript
- Git y GitHub Pages

##Funcionalidades
- Navegación fija con menú hamburguesa
- Diseño responsivo
- Tarjetas informativas de platillos típicos
- Formulario con validación usando JavaScript
- Publicación en GitHub Pages
